# Проект платформы для онлайн-обучения на DRF
## Описание:
Данный проект является back-end версии для платформы онлайн-обучения
## Содержание:
* модуль manage.py в корне проекта
* директория config
* директория materials
* директория users
* папка static
* фикстура для модели Payment

Приложение содержит разрешения для владельцев курсов/уроков и разрешения для группы модераторов.

Для JWT-авторизации использован djangorestframework-simplejwt.

Оплата курсов возможна через [Stripe](https://docs.stripe.com/api)

Управление отложенными и периодическими задачами реализованно с помощью ceelry и celery beat

## Запуск проекта через Docker Compose
Для запуска сервисов одной командой проект использует Docker Compose. 
### Состав проекта:
1. Web-приложение на Django
2. PostgreSQL - БД
3. Redis - брокер сообщений для Celery
4. Celery Worker - для фоновых задач
5. Celery Beat - для периодических задач
### Действия, необходимые для подготовки к запуску проекта:
1. Установить Docker и Docker Compose
2. Проверить, что Docker запущен
3. Запусть Redis
### Запуск проекта:
1. Склонируйте репозиторий
```
git clone https://github.com/knewtr/drf-online-school
```
2. Создайте файл .env в корне проекта и добавьте в него переменные окружения аналогично файлу .env.sample
3. Запустите проект с помощью команды
```
docker-compose up -d
```
После запуска приложение будет доступно по этому адресу http://localhost:8000/
### Команды для проверки работоспособности сервисов:
* Просмотр зарущенных контейнеров:
```
docker-compose ps
```
* Просмотр логов всех контейнеров:
```
docker-compose logs
```
* Остановка сервисов и удаление контейнеров:
```
docker-compose down
```
